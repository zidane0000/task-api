## Getting Started

### Prerequisites

- Go 1.23

## Project Structure

```text
task-api/
├── cmd/server/          # Main application entry point
├── internal/
│   ├── handlers/        # HTTP handlers/controllers
│   ├── models/          # Data models and structs
│   └── storage/         # Data storage layer
├── tests/               # Test files
└── docs/                # Project documentation
```
